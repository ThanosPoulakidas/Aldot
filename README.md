# Aldot

Aldot is a display typeface designed by **Thanos Poulakidas**. The project explores geometric forms and contemporary letter construction while maintaining readability and a distinctive visual identity.

## About

Aldot is intended primarily for headlines, posters, branding, and editorial applications. Its design combines modern proportions with clean outlines, making it suitable for both digital and print use.

The project is developed as an open-source font family and is licensed under the SIL Open Font License v1.1.

## Character Set

Current support includes:

* **GF Latin Core and Plus**
* **GF Greek Core and Plus**
* **GF Cyrillic Core**
* Basic punctuation
* Numerals
* Symbols
* Alternative Greek lowercase forms

Additional language support and glyph coverage may be expanded in future releases.

## Building

Source files are located in the `sources/` directory.

Compiled font binaries are located in the `fonts/` directory.

The repository follows the upstream structure recommended by Google Fonts.
